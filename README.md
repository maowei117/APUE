## 为什么创建这个项目
此项目为 《UNIX环境高级编程》 的课后习题个人答案，这种偏实际应用的书，需要配合每一章后面的习题使用效果更佳。

每一章的习题答案放在独立的目录中,代码以习题号命名,例如第一章第一题的代码(如果有的话)将命名为1_1.c

操练起来!
